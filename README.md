 # 安裝手冊

## 系統需求
- 作業系統：Windows, macOS, Linux
- Node.js 版本：14.x 或以上
- npm 版本：6.x 或以上

## 安裝步驟

1. 下載並安裝 [Node.js](https://nodejs.org/) 和 npm。
2. 開啟終端機並執行以下指令以確認 Node.js 和 npm 是否安裝成功：
    ```sh
    node -v
    npm -v
    ```
3. 將專案從版本控制系統中克隆到本地端：
    ```sh
    git clone <repository-url>
    ```
4. 進入專案目錄：
    ```sh
    cd <project-directory>
    ```
5. 安裝專案所需的相依套件：
    ```sh
    npm install
    ```
6. 執行專案：
    ```sh
    npm start
    ```
